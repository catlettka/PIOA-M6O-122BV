│       └── tui.py

│   ── test\_memory.py

---



## Файл memory.py



Файл `memory.py` содержит всю логику базы данных.



### Реализовано:



#### Структура базы данных

- База данных на основе `dict`

- Поддержка нескольких таблиц

- Переключение между таблицами



#### Работа с таблицами:

- `create\_table` — создание таблицы

- `use\_table` — выбор активной таблицы

- `list\_tables` — список таблиц

- `get\_table` — получение текущей таблицы



####CRUD операции:



- CREATE — `create\_record`

- READ — `select\_record`

- UPDATE — `update\_record`

- DELETE — `delete\_record`



#### Дополнительно:

- сортировка по полю (ASC / DESC)



---



## Файл tui.py



Файл `tui.py` реализует консольный интерфейс (TUI).



### Функции:

- вывод меню

- ввод данных

- обработка ошибок

- вызов backend функций



### Команды:

- 1. Создать таблицу

- 2. Добавить запись

- 3. Показать записи

- 4. Обновить запись

- 5. Удалить запись

- 6. Список таблиц

- 7. Переключить таблицу

- 8. Сортировка

- 0. Выход
