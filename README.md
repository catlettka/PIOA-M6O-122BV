# Описание структуры проекта



Проект реализует in-memory базу данных.



```text

PIOA-ГРУППА/

│

├── src/

│   └── db/

│       ├── backend/

│       │   ├── _init_.py

│       │   ├── memory.py

│       │   └── errors.py

│       │

│       ├── _init_.py

│       ├── main.py

│       └── tui.py

│

├── tests/

│   ── test\_memory.py

│     ├──  \_init\_.py

│

├── .gitignore

└── README.md

```

---


## Структура проекта

- `src/db/backend/memory.py` — логика базы данных (Table, Database)
- `src/db/backend/errors.py` — обработка всех исключений
- `src/db/tui.py` — консольный интерфейс пользователя
- `src/db/main.py` — точка входа
- `tests/test_memory.py` — автоматизированные тесты

---

## Основные возможности

### Работа с таблицами
- создание таблиц
- переключение между таблицами
- просмотр списка таблиц

### CRUD операции
- добавление записей
- просмотр записей
- обновление записей
- удаление записей

### Дополнительно
- сортировка по полю (по возрастанию / убыванию)
- фильтрация данных


---

## Обработка ошибок

Реализованы пользовательские исключения:
- таблица не создана
- таблица уже существует
- пустая таблица
- некорректное поле
- ошибка типа данных
- ошибка валидации

---

## Тестирование

Тесты покрывают:
- CRUD операции
- сортировку
- фильтрацию
- работу с таблицами
- обработку ошибок

## Запуск:

Через терминал

1. Перейти в папку src: cd src
2. Запустить программу: py -m db
