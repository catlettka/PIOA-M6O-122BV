# Описание структуры проекта

Проект реализует in-memory базу данных

```text
PIOA-НомерГруппы/

│

├── src/

│ └── db/

│   └── backend/

│   │  ├── _init_.py 

│   │  └── memory.py   логика базы данных (таблицы + CRUD)

│   │

│   ├── _init_.py

│   ├── _main_.py   точка входа (запуск программы)

│   └── tui.py        текстовый пользовательский интерфейс

│

└── README.md
```

---


## Файл memory.py содержит всю логику базы данных.

### Реализовано:

1. структура базы данных через dict позволяющая работать с несколькими таблицами
2. работа с таблицами:
- создание таблицы (create_table)
- выбор таблицы (use_table)
- список таблиц (list_tables)

3. CRUD операции для таблиц:

- `create_record` — добавление записи
- `select_record` — выборка записей
- `update_record` — обновление записи
- `delete_record` — удаление записи

---

## Файл tui.py реализует консольный интерфейс 

### Функции:

- вывод меню
- ввод данных пользователя (input)
- обработка ошибок
- вызов функций из memory.py

1. Поддерживает команды:
2. Добавить запись
3. Показать все записи
4. Найти записи
5. Обновить запись
6. Удалить запись
7. Создать таблицу
8. Выбрать таблицу
9. Показать таблицы
0. Выход

Файл _main_.py точка входа в программу.

---

## Описание функциональности проекта

1. Работа с таблицами
- создание новой таблицы
- выбор активной таблицы
- просмотр списка таблиц

2. CRUD операции

- CREATE - добавление новой записи студента
- READ - вывод всех записей; поиск по фильтрам: id, имя, фамилия, возраст, пол
- UPDATE - изменение существующей записи по id; можно менять отдельные поля
- DELETE - удаление записей: по id; по фильтру

---

## Инструкция по запуску проекта

1. Через терминал
2. Перейти в папку src: cd src
3. Запустить программу: py -m db
